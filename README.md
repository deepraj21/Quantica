# Quantica

## Installation Guide

### client

```bash
cd client
```

```bash
npm install
```

```bash
npm run dev
```

### server

```bash
cd server
```

```bash
python -m venv venv
```

```bash
./venv/scripts/activate
```

```bash
pip install -r requirements.txt
```

```bash
python app.py or flask run
```
